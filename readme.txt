# PIDataReaderApps

This folder contains a Visual Studio solution named PIDataReaderApps.
The solution is meant to include all projects needed to build PIDataReader executables.

Usage instructions.

1. On a Windows machine, create a top level folder [L0_FOLDER]
2. Within [L0_FOLDER] create one folder named PIDataReaderLib
	2.1 cd to PIDataReaderLib
	2.2 connect to GitHub and pull PIDataReaderLib
3. Within [L0_FOLDER] create one folder named PIDataReaderApps
4. cd to PIDataReaderApps
5. Create five folders
	5.1 PIDataReaderCommons
		5.1.1 cd to PIDataReaderCommons
		5.1.2 connect to GitHub and pull PIDataReaderCommons
	5.2 PIDataReaderExe
		5.2.1 cd to PIDataReaderExe
		5.2.2 connect to GitHub and pull PIDataReaderExe
	5.3 PIDataReaderSvc
		5.3.1 cd to PIDataReaderSvc
		5.3.2 connect to GitHub and pull PIDataReaderSvc
	5.4 PIDataReaderSetup
		5.4.1 cd to PIDataReaderSetup
		5.4.2 connect to GitHub and pull PIDataReaderSetup
	5.5 PIDataReaderSolution
		5.5.1 cd to PIDataReaderSolution
		5.5.2 connect to GitHub and pull PIDataReaderSolution
6. Use Visual Studio to open the solution file found in PIDataReaderSolution
